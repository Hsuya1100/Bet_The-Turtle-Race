# Bet_The-Turtle-Race
simple python based turtle race created using turtle module .
Bet over our turtle....
## How to Begin the Race:

### Clone this repo
```
git clone https://github.com/Hsuya1100/Bet_The-Turtle-Race.git
```
### Change the working directory
```
cd Bet_The-Turtle-Race
```
### Begin the race
```
python3 race.py
```
